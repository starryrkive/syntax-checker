# Syntax Checker

This project implements a basic syntax checking tool using lexical analysis and parsing techniques. The program reads input code, analyzes tokens, and validates the syntax using defined grammar rules.

The project demonstrates core compiler design concepts such as token generation and syntax validation.

---

## Concepts Used

- Lexical Analysis
- Tokenization
- Parsing
- Grammar Rules

---

## Technologies

Language
- C

Tools
- Lex / Flex
- Yacc / Bison

---

## Features

- Reads source code input
- Generates tokens through lexical analysis
- Parses tokens based on defined grammar rules
- Detects syntax errors in the input code
- Displays error messages for invalid syntax

---

## Project Structure

syntax-checker  
│  
├── lexer.l        Lex file for token generation  
├── parser.y       Yacc file containing grammar rules  
├── main.c         Driver program  
└── README.md

## Installation and Execution

- Navigate into the Project directory
- Generate the lexer and parser files
- Compile the program
- Run the program
